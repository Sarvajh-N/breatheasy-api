BreatheEasy API

Lung cancer screening eligibility + referral PDF generation

This API implements the USPSTF 2021 Grade B Recommendation for lung cancer screening. It checks eligibility, returns billing codes (CPT/ICD-10), and generates physician referral PDFs. Bilingual (English/Spanish). No data stored. Free to use.

Endpoints

POST /api/eligibility

Check if a patient meets USPSTF 2021 screening criteria. Returns eligibility result + billing codes.

Request Body

{
  "age": 55,
  "cigarettes_per_day": 20,
  "years_smoked": 30,
  "quit_years": 5,
  "lang": "en"
}

Response (Eligible)

{
  "eligible": true,
  "pack_years_calculated": 30,
  "billing_codes": {
    "cpt": "71271",
    "icd10": ["Z87.891", "Z12.2"],
    "modifier": "33"
  },
  "cost": "$0 under ACA"
}
POST /api/referral

Generate a physician referral PDF with CPT/ICD-10 billing codes. Only works for eligible patients.

Additional Fields

{
  "format": "base64"  // or "binary" (default — returns PDF file)
}
GET /api

API documentation — returns all endpoints, guidelines, and billing codes as JSON.

Integration Example

For any website — one fetch call:

fetch('https://breatheasy-api.vercel.app/api/eligibility', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    age: 55,
    cigarettes_per_day: 20,
    years_smoked: 30,
    quit_years: 5
  })
})
.then(r => r.json())
.then(data => {
  if (data.eligible) {
    alert('Eligible! CPT: ' + data.billing_codes.cpt);
  }
});
